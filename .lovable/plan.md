

## Система поиска и рекомендаций раскидок

### Что будет сделано

Расширенная система поиска, фильтрации и рекомендаций чужих раскидок на странице карты. Пользователь сможет искать раскидки по тексту, фильтровать по множеству параметров и кликать по области на карте, чтобы найти раскидки рядом с этой точкой.

### Компоненты

**1. Панель поиска и фильтров** (`src/components/GrenadeSearchPanel.tsx`)
- Текстовый поиск по названию и описанию раскидки (debounced input)
- Существующие фильтры (тип гранаты, команда, избранное) переносятся в эту панель
- Новые фильтры: сложность (easy/medium/hard), тип броска (standing, jump_throw и т.д.)
- Сортировка: по дате, по популярности (кол-во добавлений в избранное)
- Сворачиваемая панель, чтобы не занимать место на мобильных

**2. Поиск по области на карте**
- Режим "поиск по области": пользователь кликает на карту, и система показывает раскидки, у которых точка приземления (landing point) находится в радиусе ~5% от клика
- Визуальный индикатор радиуса поиска на карте (полупрозрачный круг)
- Кнопка для сброса зоны поиска

**3. Список результатов** (`src/components/GrenadeSearchResults.tsx`)
- Карточки раскидок под картой с превью: название, тип, сложность, автор
- Клик по карточке открывает GrenadeFullInfo (как сейчас)
- Показ количества найденных результатов

**4. Рекомендации**
- Секция "Похожие раскидки" в GrenadeFullInfo — раскидки с близкой точкой приземления на той же карте
- Секция "Популярные раскидки" на странице карты — топ по добавлениям в избранное

### Изменения в базе данных

Нужна view или запрос для подсчета количества добавлений в избранное по каждой раскидке (для сортировки по популярности). Это можно сделать на клиенте без миграции, так как данные favorites уже доступны.
